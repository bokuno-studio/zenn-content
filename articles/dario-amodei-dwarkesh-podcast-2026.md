## はじめに

Anthropic CEO の Dario Amodei が Dwarkesh Patel の Podcast に再登場。前回対談から3年を経て、AI 技術の進展・エージェント経済の到来・AGI ガバナンスの難題まで2時間超にわたり深掘りした対話です。

## 1. AI技術の進展：予測との比較

![Dario Amodeiが語るAI技術の進展](/images/screenshot-dario-01.jpg)

> 「技術の指数関数的成長は、おおむね予測通りに進んでいる」

**RL スケーリングの発見**が最大の変化です。事前学習と同様のスケーリング則が RL（強化学習）でも成立することが確認されました：数学コンテスト（AIME）その他の RL タスクで、学習時間に対して線形にスコアが向上するという結果が各社から報告されています。

**2035年までの確信度は90%**。唯一の不確実性は「検証不可能なタスク」（火星ミッション計画・CRISPR 発見のような根本的科学的発見・小説執筆）にあるとしています。

### 人間の学習とモデルの学習の比較

LLM の学習には人間に対応するものがない独自のフェーズがあります：

| フェーズ | 人間 | LLM |
|---------|------|-----|
| 長期学習 | 数ヶ月〜年 | 事前学習（事後は固定） |
| 短期学習 | 数時間〜日 | In-context learning |
| 反射 | ミリ秒 | 推論（inference） |
| 継続学習 | 毎日の経験蓄積 | **現在は存在しない** |

## 2. Claude Code とエージェントの現実

![Dario Amodeiが語るClaudeとエージェント](/images/screenshot-dario-02.jpg)

> 「ループを閉じる能力——自己完結したシステムでのフィードバックサイクル——が、どれほどの生産性向上をもたらすかを考えさせられる」

Claude Code は大企業でも個人開発者と同じスピードで導入できるはずですが、実際の普及は組織の複雑さによって遅れています。Dario は「技術の普及は以前の技術より速い。しかし無限に速いわけではない」と強調します。

### チューリングテストを超えているのに自動化が進まない理由

「1時間話すと一般的に論理的な人間と見分けがつかないシステムが、なぜホワイトカラーの仕事を自動化しないのか」——Dwarkesh のこの問いに対して Dario は：

1. **継続学習の欠如**: モデルは「コードベースを6ヶ月かけて理解する」という人間の学習を in-context で素早く行えるが、その理解を保持できない
2. **ループを閉じられない**: 自己完結したフィードバックサイクルがなければ、エージェントは「指示待ち」状態になる
3. **検証の難しさ**: 優秀な開発者を使っても生産性向上が測定できない事例がある（2024年の研究では、経験豊富な開発者が AI を使って主観的に生産性が上がったと報告したが、実測値は変わらなかった）

## 3. 「天才の国」という概念

> 「天才が集まった国のような計算能力をデータセンターに詰め込む。それが今後の形だ」

Dario の「天才の国（Country of Geniuses）」概念：

- **現在の方向性**: 単一の超巨大モデルより、多数の賢いエージェントが並列で動くシステムへ
- **コードベース**: Claude Code がコードベース全体を in-context で理解できるなら、6ヶ月の新人期間を圧縮できる
- **制約**: 現在は継続学習がないため、各タスクで「ゼロから」始まる

## 4. AI経済学：コストと収益の均衡

> 「各モデルは利益を生んでいる。しかし会社は損している。次の訓練に指数関数的な投資が続くから」

Dario が示した業界の経済モデル（概念的な例）：

```
年間100B$のコンピュートを投資
→ 150B$の収益（推論は50%超の粗利）
→ 50B$の利益

しかし次世代モデルの訓練に100B$以上が必要
→ 指数的スケールアップで会社は赤字
→ 均衡点はモデル訓練の指数的成長が止まるとき
```

**均衡状態（将来のシナリオ）**:
- 推論コンピュートのほぼ全額を収益でカバーできる
- 訓練スケールアップが安定
- そこで初めて「天才の国」が安定的に稼働する

## 5. 継続学習（Continual Learning）の欠如

> 「継続学習がなければ、本当の意味での世界変革は難しいと少し懐疑的だ」

現在のモデルは推論時に学習しません。Dwarkesh が指摘するように、**6ヶ月働いた AI アシスタントと今日初めて会った AI アシスタントを区別できない**状況が続いています。

Dario の見解：
- 「2〜3年以内に解決されるはずの問題」
- チューリングテストを超えていること・継続学習がないことが同時に真実である奇妙な状況
- 「この問題が解決されるまで、白紙状態の追加分野が残り続ける」

## 6. 恩恵の普及タイムラインと格差

> 「ポリオワクチンは50年経っても世界の最貧困地域への普及が完全ではない。AI も同じリスクがある」

ラボで AGI が実現してから「すべての人に恩恵が届く」までの時間：

| フェーズ | 推定期間 |
|---------|---------|
| ラボ実現 | 基準点 |
| 企業での導入 | 数ヶ月〜数年 |
| 発展途上国への普及 | 最も困難 |
| 医療（全世界） | 10〜20年（楽観的推測） |

Dario の立場：「恩恵の普及はほとんどのケースで経済的インセンティブが働くため、ポリオほど遅くはならない。しかし意識的な努力が必要」

## 7. バイオリスク：AI×テロリズムの懸念

> 「今年後半に『AIバイオテロリズムは深刻な脅威』と判断せざるを得ない世界が来るかもしれない」

Dario が最も強調した安全リスク：

- **脅威の性質**: AI が生物兵器の設計・改良に使われるリスク
- **対応**: 分類器の義務付け・連邦標準の設定を視野に入れている
- **知的誠実さ**: 「リスクはまだ明らかにはなっていない。しかし進展のペースを考えると、今のうちに備える必要がある」

## 8. AIガバナンスと地政学

> 「民主主義国家の連合が、権威主義国家を含む世界とルールを交渉する形が理想。一国だけがルールを決めるべきではない」

### AI と権威主義

- AI が権威主義政府を強化するリスクを認識しつつ、技術の恩恵を世界に広く届けることを重視
- 「歴史的に、技術の普及はプラスサムだった。AI も同様と考えるが、難しい問題だ」

### AI コンスティテューション（AI の価値体制）

Anthropic が採用する多層フィードバックモデル：

1. **企業レベル**: Anthropic がコンスティテューションを設定・公開し、コミュニティがコメント
2. **業界レベル**: 複数企業が異なるコンスティテューションを公開し、比較・競争
3. **政府レベル**: 政府がメタルールを設定
